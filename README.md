# 空军主力战机 - 改进版网站

基于你原有的战机图库网站，添加了多页面导航、图片展示、搜索和鼠标跟踪等功能。

## 🎯 新增功能

✅ **多页面导航** - 4个完整页面（首页、图片库、战机详情、关于）  
✅ **图片库** - 点击图片可放大查看，支持键盘导航（←→键）  
✅ **战机详情页** - 显示完整的技术参数和发展历史  
✅ **鼠标跟踪效果** - 自定义光标跟随动画  
✅ **保留原有设计** - 保持你的深色主题和现代化风格  
✅ **搜索功能增强** - 保留并优化了原有的搜索功能

## 🚀 部署到 GitHub Pages

### 方法1：替换现有仓库文件

1. 下载所有文件到本地
2. 用新文件替换你仓库 `antushenguzi/web-work` 中的文件
3. 提交并推送到GitHub：

```bash
git add .
git commit -m "添加多页面导航和新功能"
git push origin main
```

### 方法2：使用GitHub网页上传

1. 进入你的仓库 https://github.com/wanna-aa/web-work
2. 点击 "Add file" → "Upload files"
3. 上传所有HTML、CSS、JS文件
4. 点击 "Commit changes"

## 🖼️ 关于图片

当前使用的是占位图片（placeholder）。你需要：

1. 将你的战机图片放到仓库的 `images/` 文件夹中
2. 在HTML文件中将占位符URL替换为你的图片路径

**替换示例：**

```html
<!-- 原来的占位符 -->
<img src="https://via.placeholder.com/600x450/1a2332/4ea1ff?text=F-22" alt="F-22">

<!-- 替换为你的图片 -->
<img src="images/f22.jpg" alt="F-22">
```

## 🎯 使用技巧

### 搜索功能
在搜索框中输入关键词，如：
- 国家：`美国`、`中国`、`俄罗斯`
- 型号：`f-22`、`j-20`、`su-57`
- 类型：`隐身`、`舰载`、`战斗机`

### 图片库
1. 点击任意图片放大查看
2. 使用键盘 ← → 键切换图片
3. 点击 X 或按 ESC 键关闭

### 战机详情
- 在首页点击战机卡片查看详情
- 或直接访问：`details.html?id=战机代号`
- 支持的ID：f22, f35, j20, su57, 等

## 🔧 自定义

### 修改颜色主题

在 `styles.css` 的开头修改CSS变量：

```css
:root {
    --bg: #0b0f14;          /* 背景色 */
    --accent: #4ea1ff;      /* 主题色 */
    --text: #e7eef7;        /* 文字颜色 */
}
```

### 添加更多战机

在 `index.html` 中复制卡片代码并修改内容。

### 添加战机详情

在 `script.js` 的 `aircraftData` 对象中添加数据。
